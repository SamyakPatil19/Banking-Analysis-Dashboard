# Banking-Analysis-Dashboard
#  Project Overview

This project focuses on risk analytics in the banking and financial services domain, demonstrating how data can be used to minimize lending risk and support better loan approval decisions.
An interactive Power BI dashboard was built using banking client data provided in Excel format.

The dashboard helps banks analyze customer profiles to determine whether an applicant is likely to repay a loan, enabling smarter and data-driven decisions.

#  Tools & Technologies Used

Microsoft Excel – Data source

Power BI – Data modeling, DAX, and visualization

DAX – KPI and measure creation

#  Data Cleaning & Feature Engineering

The following transformations were performed in Power BI:

Created Engagement Timeframe to track client association duration

Calculated Engagement Days using joining date and current date

Created Income Bands (Low / Mid) using estimated income bins

Added Processing Fees based on fee structure categories

# Calculated Measures (DAX)

Key DAX functions used:

SUM,
DISTINCTCOUNT,
SUMX,
SWITCH,
DATEDIFF

Important KPIs created:
Total Clients,
Total Loan,
Bank Loan,
Business Lending,
Total Deposits,
Total Fees,
Credit Card Balance,
Savings, Checking & Foreign Currency Accounts,
Engagement Length

#  Dashboard Pages

The Power BI report includes:

1. Home Dashboard – Overall banking overview

2. Loan Analysis – Loan distribution and risk insights

3. Deposit Analysis – Customer deposit patterns

4. Summary Dashboard – Consolidated KPIs and insights

# Key Insights

Private banks have a higher number of clients compared to others

Loan distribution varies significantly by customer profile and income band

Engagement duration strongly correlates with deposits and loan activity

Certain nationalities and customer segments hold higher loan exposure
